# Common Web Ports
# Source: Practical experience summary

## Standard Web Ports
- 80    - HTTP (default)
- 443   - HTTPS (default)

## HTTP Proxy Ports
- 8080  - HTTP proxy / alternative HTTP
- 8888  - HTTP proxy (common)
- 3128  - Squid proxy

## HTTPS Proxy Ports
- 8443   - HTTPS proxy / alternative HTTPS
- 9443   - HTTPS proxy

## Development Ports
- 3000   - Node.js/React development
- 3001   - Node.js development
- 4000   - Flask/Python development
- 5000   - Flask/Python development
- 5001   - Flask development
- 5050   - Various frameworks
- 6000   - Various frameworks
- 7000   - Various frameworks
- 8000   - Django/IIS/HTTP proxy
- 9000   - PHP-FPM/HTTP proxy
- 6060   - Java applications
- 7070   - Java applications
- 9090   - Java applications / Management panel
